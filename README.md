# 1RGB

1RGB... pfft. Just another RGB controller app.

Or is it?

It's not.

Just don't question it, okay...?

## The Project Story
*(TL;DR: I hate Razer Synapse and OpenRGB doesn't have enough effects.)*

So I was using my Razer keyboard (BlackWidow V4 X to be exact) and suddenly had that irresistible urge to make the keyboard light up however I wanted instead of using the default presets.

So I downloaded Razer Synapse for macOS.

And guess what?

**IT DOESN'T SUPPORT MY KEYBOARD?!?**

So now I have a mission.

Develop this app for my sake... and everyone else's.

And welcome to 1RGB.
Really, welcome.

## Features

- Custom RGB effects
- User-created effects
- Physics-based lighting simulations
- Game integration support *(planned)*
- Cross-platform architecture
- Extensible plugin system
- An ecosystem that may or may not make you question your decisions

## Built With

- C
- Assembly
- Lua
- Python
- Swift *(macOS build)*
